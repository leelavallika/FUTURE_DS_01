# Business Sales Performance Analytics Dashboard

## Project Overview

This project analyzes retail sales data to identify business trends, top-performing categories, regional performance, and sales patterns. The analysis was performed using Python and Power BI.

## Tools Used

- Python
- Pandas
- Matplotlib
- Power BI Desktop

## Dataset

Sample Superstore Dataset

## Key Business Insights

- Technology generated the highest sales.
- West region achieved the highest revenue.
- South region recorded the lowest sales.
- November was the best-performing month.
- Canon imageCLASS 2200 Advanced Copier was the highest-selling product.
